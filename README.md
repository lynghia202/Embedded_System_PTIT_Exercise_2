# Embedded_System_PTIT_Exercise_2

## Giới thiệu
Bài tập 2 - Blinking LED

## Phần cứng sử dụng
- STM32F103C8T6
- Cảm biến nhiệt độ DS18B20
- Module UART-USB

## Cách chạy
1. Mở project bằng KeilC (hoặc STM32CubeIDE).
2. Nạp code vào board.
3. Quan sát kết quả qua Serial Monitor.
